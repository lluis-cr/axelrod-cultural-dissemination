# Problem Definition

This project investigates how simple local interaction rules can generate
large-scale cultural convergence or fragmentation.

The baseline setting follows Axelrod's cultural dissemination model: agents are
described by multiple cultural features, neighbours interact more often when
they are already similar, and interaction makes them even more similar. The
project asks when this process produces:

- a single shared culture,
- several stable cultural regions,
- persistent diversity,
- or long-lived polarised structures.

The broader motivation is that many social systems display both integration and
division at the same time. Axelrod's model provides a minimal framework for
studying how those outcomes can emerge from decentralised interaction.
