# Limitations

The current repository is still an early-stage research codebase. The main
limitations of the baseline are:

1. The maintained model is limited to a regular lattice with open boundaries.
2. The baseline run length may end before the system reaches a frozen state.
3. The current package has only minimal reporting and metric infrastructure.
4. Repeated parameter sweeps are not yet automated.
5. Immigration, creator-consumer influence, and alternative topologies are
   scaffolds rather than completed implementations.
6. The current results come from a very small number of runs and should not yet
   be treated as robust empirical findings.

These limitations are acceptable for an early-stage MSc research repository,
and they define the next development priorities.
