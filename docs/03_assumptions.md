# Assumptions

The current baseline model makes the following simplifying assumptions:

1. Agents are culturally represented by a fixed-length discrete feature vector.
2. All features are equally weighted.
3. Influence is dyadic and local.
4. Similarity increases the chance of interaction.
5. Interaction always reduces difference between the selected pair.
6. The neighbourhood structure is fixed during the run.
7. Agents do not strategically rewire social ties.
8. There is no exogenous media, migration, or institutional shock in the
   baseline.
9. Time advances through asynchronous random updates.
10. Random initialisation is used unless otherwise specified.

These assumptions are intentionally minimal so the baseline can serve as a
clear reference point for richer extensions.
