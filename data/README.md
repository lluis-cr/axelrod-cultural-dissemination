# Data

This project currently does not depend on an external empirical dataset.

## What lives here
- `raw/`: reserved for any future external or manually collected inputs.
- `processed/`: reserved for transformed inputs or simulation-ready reference
  files if those are introduced later.

## Current status
The present repository is primarily simulation-based. The main outputs are
generated figures and, later, summary tables derived from Axelrod model runs.

If external data is introduced in future extensions, this folder should be used
to document provenance, processing steps, and licensing.
